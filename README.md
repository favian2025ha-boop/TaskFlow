# TaskFlow
TaskFlow – Gestor de Tareas Inteligente
Descripción del Proyecto:

TaskFlow es una aplicación web tipo SPA (Single Page Application) diseñada para ayudar a los usuarios a organizar, visualizar y gestionar sus tareas diarias de manera eficiente.

En un entorno donde la productividad personal es clave, muchas personas tienen dificultades para llevar un control claro de sus pendientes. TaskFlow busca solucionar este problema proporcionando una interfaz intuitiva que permite agregar, visualizar y filtrar tareas según su estado (completadas o pendientes), mejorando así la organización y el enfoque diario.

Objetivo del Proyecto:

Construir una aplicación frontend moderna utilizando React y Vite, aplicando buenas prácticas de arquitectura, componentización y renderizado dinámico mediante JSX.

Tecnologías Utilizadas
 React
 Vite
 Node.js / NPM
 CSS3 (Flexbox / Grid)
 JSX
